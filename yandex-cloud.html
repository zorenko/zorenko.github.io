Yandex Cloud User experience and visual design for Yandex Cloud products (Console, Partner's portal, Community)

Yandex Cloud is a a set of related services, a cloud platform from the Russian Internet company Yandex. Yandex.Cloud provides private and corporate users in the “as a service” format infrastructure services - Yandex Compute Cloud, Yandex Object Storage, Yandex Message Queue, Yandex Virtual Private Cloud and Yandex Identity and Access Management, as well as Yandex Managed Service platform services (for various databases), Yandex SpeechKit and Yandex Translate.

Yandex.Cloud is built on the basis of the company's own data centers located in Russia, as well as software solutions based on which Yandex sites and applications work - Search, Maps, Metrica and others.
